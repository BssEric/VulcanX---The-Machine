# 📖 About

**VULCAN X** is a premium landing page concept created to showcase a fictional high-end notebook through cinematic storytelling, immersive animations and modern web technologies.

Instead of presenting specifications through traditional sections, the experience guides the user through an interactive journey where every scroll reveals a new chapter of the product.

The project was designed to simulate the quality of launches from brands such as Apple, Alienware and ASUS ROG.
