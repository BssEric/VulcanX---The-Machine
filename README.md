# 📖 Sobre o projeto

O **VULCAN X** é uma landing page premium criada para demonstrar como uma apresentação de produto pode ir muito além de uma página convencional.

Inspirado em lançamentos de empresas como **Apple**, **Alienware**, **ASUS ROG** e **Razer**, o projeto transforma a navegação em uma experiência imersiva, utilizando animações sincronizadas ao scroll, vídeos, transições cinematográficas e uma interface sofisticada.

Cada seção foi desenvolvida para contar uma história sobre o produto, destacando seu design, desempenho e exclusividade.
